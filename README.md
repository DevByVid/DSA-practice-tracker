# DSA Practice Tracker

A simple tracker to log DSA problems, track progress, and visualize streaks.

## Features

- Add problems with date, topic, difficulty, and status
- Delete problems
- Filter by Solved / Unsolved / All
- Stats: total problems, solved, unsolved
- Progress bar showing completion percentage
- Current streak counter
- Dark mode toggle
- Fully responsive and mobile-friendly

## How to Use

1. Open `index.html` in your browser
2. Fill the form and add problems
3. View stats, progress, and streaks
4. Filter problems by status

## Tech Stack

- HTML5
- CSS3
- JavaScript
- LocalStorage
